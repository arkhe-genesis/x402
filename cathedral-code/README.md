# Cathedral Code

Cathedral Code is a multi-agent coding platform combining the sovereignty and verifiable auditing of Cathedral Arkhe with the extensibility and multi-agent capabilities of OpenCode and Remix Project.

## Components
- `cathedral-server`: The main orchestrator and API.
- `cathedral-remix-bridge`: Integration with Remix runtime.
- `cathedral-opencode-bridge`: Integration with OpenCode.
- `cathedral-core`, `cathedral-identity`, `cathedral-wormgraph`, `cathedral-zk`: Core blockchain and verifiable components.
